## Форматирование

Список:
- *italic*
- **bold**
- ~~srikethrough~~
- ==highlight==
- ^superscript^
- ~subscript~
- `code`

Пропадающие спецсимволы: x*y*z

Код:

	x*y*z
	line 2

То же самое другим способом:

```
x*y*z
line 2
```
